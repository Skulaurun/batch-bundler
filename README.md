# batch-bundler
Batch bundler is a single file project. Used for bundling multiple batch files to one `bundle.bat` file. <br>
Warning: This project is <b>WORK IN PROGRESS</b>.
## Usage
`bbat.py [-i I] <mainFilePath> [-o O] <outputDirectory>`
